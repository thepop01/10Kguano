### Game Features
- **Controls**: Keyboard (Arrow/A/D) + Touch
- **Difficulty**: Hard mode with progressive scaling
- **Birds**: 1-5 birds based on score
- **Win Condition**: 10,000 points
- **Health System**: 3 splash hits allowed
- **Responsive**: Works on desktop and mobile
